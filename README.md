# The Commonplace

Personal website of Hiran Venugopalan — product designer based in Bengaluru, India.

Built with [Hugo](https://gohugo.io) and deployed to [hiran.in](https://hiran.in) via GitHub Pages.

Contains writing, reading notes, bookmarks, and a newsletter archive. Notes are written in Obsidian and published selectively.
