Editing YAML, JSON, or hundreds of markdown files in an Obsidian vault in an IDE works, but it isn't comfortable. Marktable turns that data into a table view. Add columns, check consistency, fill gaps, with the ease of a spreadsheet while the source stays in markdown files.

The idea came from managing structured data across a vault and finding no simple visual layer for it. The developer is building, the product thinking is ongoing. Still early.
