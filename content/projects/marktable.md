Any list managed as a JSON or YAML file is technical by nature. Checking consistency across entries, making sure every item has the same fields in the same order, filling gaps, doing any of this in a code editor is more effort than it should be. A spreadsheet is the right mental model for this kind of work but spreadsheets don't work on these files. Marktable turns them into a live table view where you can edit, sort, and make things consistent while the source files stay exactly as they are. The same need came up with markdown frontmatter in Obsidian, so we made one tool that handles both. Still in alpha.
