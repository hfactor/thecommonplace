---
title: Links should be earned by the sentence
date: 2026-06-16
---

A link should appear because the sentence needs the other idea. When the link is added to satisfy the system, the note starts performing connectedness instead of thinking. The sentence becomes weaker because it is carrying a relationship it has not earned. [[Non-linear note making]] only works when the connection changes what the note can do.

This matters more as the archive grows. A dense graph can still be a shallow one if the edges are polite rather than necessary. [[Evergreen notes should be atomic]] gives each note a clear claim, but the link has to respect that claim instead of stretching it toward every neighbour. A quiet note with one honest connection is more useful than a busy note with four decorative ones.
